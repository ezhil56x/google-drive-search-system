## Setting up the project

### Frontend

1. Go to the frontend directory

```bash
cd frontend
```

2. Copy the `.env.example` file to `.env`

```bash
cp .env.example .env
```

3. Install the dependencies

```bash
npm install
```

4. Start the development server

```bash
npm run dev
```

### Backend
